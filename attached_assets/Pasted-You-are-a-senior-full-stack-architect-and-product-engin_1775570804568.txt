PRODUCT GOAL:
Guest Pro is a hotel guest web app optimized for phone usage.
Guests can log in with a guest key and instantly access a premium AI concierge/chat interface.
The hotel manager can log in and create new guests with a generated guest key.

CURRENT MVP SCOPE:
ROLES:

2) GUEST

MANAGER FLOW:

- Manager dashboard should be minimal for now

- Clicking it opens a minimal premium form/modal/drawer:
  - First name
  - Last name
  - Room number
- On submit:
  - create guest
  - generate unique guest key
  - store everything in database
  - show the key on screen with copy button

GUEST FLOW:
- Guest logs in using guest key only
- After login show:
  - first name
  - last name
  - room number
